# Justin Li - Software Engineer Portfolio

A modern, responsive portfolio website showcasing projects, skills, and professional experience.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Featured Projects Carousel**: Interactive carousel displaying featured projects
- **3D Project Cubes**: Flip cards with hover effects to reveal project descriptions
- **Smooth Animations**: Typed.js animation for hero section, smooth scrolling
- **Multiple Sections**:
  - Hero with call-to-action buttons
  - Featured projects carousel
  - About section with tabbed information
  - Services section
  - Full project portfolio with modal details
  - Contact section with direct links

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, 3D Transforms)
- Vanilla JavaScript
- Typed.js for text animations
- Font Awesome for icons

## Getting Started

Simply open `index.html` in your browser to view the portfolio.

To deploy on Vercel:
1. Push this repository to GitHub
2. Create a new project on Vercel
3. Connect your GitHub repository
4. Vercel will automatically deploy

## Customization

- Update contact information in the contact section
- Modify projects in the `projectsData` array in `script.js`
- Adjust colors in `style.css` (CSS variables: `--primary-color`, `--secondary-color`)
- Edit bio and experience in the About section
