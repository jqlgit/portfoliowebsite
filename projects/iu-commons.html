iU Commons

Ideas United Internal Site for charity engagement and partner collaboration

Internal Platform

Abstract

iU Commons is a comprehensive internal platform developed for Ideas United to enhance partner and company charity engagement through the IUGives Initiative. The platform successfully handled 50+ partner user-generated content submissions and accumulated 4000+ internal user site visits, establishing itself as the central hub for charitable engagement.

Features

  • Server-side admin features and website
  • IUGives Initiative integration
  • Partner and company charity engagement tools
  • CMS with edge case handling
  • 50+ partner UGC submissions
  • 4000+ internal user site visits

Tech Stack

Vue.js Nuxt.js Ruby on Rails Rails API PostgreSQL Splunk

The Challenge

Implementing complex admin features while ensuring the CMS remained robust and user-friendly required careful planning of edge cases and comprehensive testing. The platform needed to balance powerful administrative tools with an intuitive interface for non-technical charity partners.

Building the Platform

The frontend was built with Nuxt.js, providing server-side rendering capabilities and excellent SEO for discovering charitable initiatives. Vue.js components handle dynamic user interactions, while the robust Rails API backend manages data persistence and complex business logic.

The CMS was engineered to handle diverse content types from partner organizations, with special attention to edge cases like duplicate submissions, media validation, and permission management. PostgreSQL ensures data integrity across concurrent submissions.

Impact

The platform became the centerpiece of Ideas United's internal operations, facilitating seamless collaboration between companies and charitable partners. With 4000+ site visits, iU Commons proved to be a valuable tool for coordination and engagement tracking.
